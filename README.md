# Handwritten OCR Project

Convert handwritten notes into digital text using OCR + NLP.

## Features

- Upload handwritten images
- Image preprocessing for better accuracy
- OCR text extraction
- Clean web interface
- Export text output

## Tech Stack

- Python
- FastAPI
- OpenCV
- Tesseract OCR
- HTML/CSS/JavaScript

## Run Locally

```bash
uvicorn app.main:app --reload
